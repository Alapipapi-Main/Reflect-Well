# **App Name**: ReflectWell

## Core Features:

- Daily Journal Entry: Users can add text-based journal entries for each day, capturing their thoughts and experiences.
- Mood Selection: Users can select a mood emoji to represent their overall mood for the day.
- Entry Review: Users can review past journal entries by date, allowing them to revisit and reflect on previous experiences.
- Mood Trend Overview: Generate a mood trend graph so users can see the evolution of their feelings. It helps them find patterns.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke calmness and reflection.
- Background color: Pale cream (#F5F5DC) for a gentle and unobtrusive backdrop.
- Accent color: Light rose (#FFB6C1) to highlight key interactive elements.
- Body and headline font: 'Literata' for a literary and reflective feel.
- Use simple, clean icons for navigation and mood selection.
- Maintain a clean and organized layout to promote focus and clarity.
- Subtle fade-in and transition animations for a smooth user experience.
